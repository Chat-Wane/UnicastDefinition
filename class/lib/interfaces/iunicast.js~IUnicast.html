    IUnicast

    Extends:

    EventEmitter → IUnicast

    An interface providing easy-to-use event-like functions on top of a peer-sampling protocol. As soon as protocols register, they get an interface. They can send messages using unicast.emit('eventName', neighborId, args) and the neighbor can catch them using unicast.on('eventName', args).
